Student Attendance Management System

The Student Attendance Management System is a web-based application developed to simplify attendance tracking and academic record management. It enables administrators and faculty members to securely manage student data, monitor attendance percentages, and maintain structured records. The system includes role-based authentication and a user-friendly dashboard for efficient data handling and reporting, helping reduce manual workload and improve overall administrative efficiency. Technologies Used: HTML, CSS, JavaScript, Java, MySQL.

Food Cart

Food Cart is a full-stack food delivery web application developed to provide a seamless and efficient online ordering experience. The platform allows users to browse restaurants, explore menus, add items to a dynamic cart, and securely place orders through an intuitive interface. The application includes user authentication, real-time order management, and an admin dashboard for restaurant control. Designed with scalability and performance in mind, it ensures secure transactions and smooth navigation for both customers and vendors. Technologies Used: HTML, CSS, JavaScript, Node.js, Express.js, MongoDB.

Campus Cab

Campus Cab is a web-based cab booking platform developed to simplify ride scheduling and booking processes. The application allows users to register, log in, book rides, manage bookings, and view ride history through an intuitive and responsive interface. The system focuses on secure authentication, efficient ride management, and smooth user experience. It includes features such as booking confirmation, ride status tracking, and user-friendly dashboard access. Technologies Used: HTML, CSS, JavaScript (Frontend), Node.js & Express.js (Backend), MongoDB (Database).
